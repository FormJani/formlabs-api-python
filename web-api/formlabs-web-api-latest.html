date__lt
string <date-time>

Filter by date time less than date time specified (ISO 8601 Format)

machine_type_id
Array of strings
Items Enum: "FORM-1-0" "FORM-1-1" "FORM-2-0" "FORM-2-1" "FORM-3-0" "FORM-3-1" "FORM-3-2" "DGJR-1-0" "FRML-3-0" "FRBL-3-0" "FRMB-3-0" "DGSR-1-0" "FRMB-3-1" "PILK-1-0" "SIFT-1" "UNKNOWN" "PILK-1-1" "SIFT-1-0" "SIFT-1-1" "CURL-1-1" "CURL-1-0" "WSHL-1-0" "FS30-1-0" "FS30-1-1" "FORM-4-0"

Filter by machine type id.

material
string
name
string

Filter by name of the print (Substring Match)

page
integer

A page number within the paginated result set.

per_page
integer

Number of results to return per page.

printer
string

Filter by printer serial

status
string

Filter by status of the print. Possible values are:
* QUEUED - Queued
* PREPRINT - Preprint
* PRINTING - Printing
* PAUSED - Paused
* FINISHED - Finished
* ABORTING - Aborting
* ABORTED - Aborted
* ERROR - Error
* WAITING_FOR_RESOLUTION - Waiting for Resolution
* PREHEAT - Preheat
* PRECOAT - Precoat
* POSTCOAT - Postcoat
